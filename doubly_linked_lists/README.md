# Doubly Linked Lists

This project covers the implementation and manipulation of doubly linked lists in C.

## Learning Objectives

- Understand doubly linked list data structures
- Traverse a doubly linked list
- Add and delete nodes
- Manage memory correctly
- Work with `prev` and `next` pointers

## Requirements

- Ubuntu 20.04 LTS
- gcc with flags:
  - Wall
  - Werror
  - Wextra
  - pedantic
  - std=gnu89
- Betty style compliance
