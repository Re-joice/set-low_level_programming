# 0x07. C - Even more pointers, arrays and strings

This project is part of the **ALX Low Level Programming** curriculum.

It focuses on deeper understanding of:
- Pointers
- Memory manipulation
- Arrays
- Strings

## Learning Objectives

At the end of this project, you should be able to explain:

- What pointers are and how they work
- How to manipulate memory in C
- How to work with arrays and strings
- The difference between pointers and arrays
- How standard memory functions like `memset`, `memcpy`, and `strchr` work
