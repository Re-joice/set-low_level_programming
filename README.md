# 0x09. Static Libraries

This project is about creating a static library in C.

The library libmy.a contains implementations of common C functions such as:
_putchar, _islower, _isalpha, _abs, _isupper, _isdigit, _strlen, _puts,
_strcpy, _atoi, _strcat, _strncat, _strncpy, _strcmp, _memset, _memcpy,
_strchr, _strspn, _strpbrk, and _strstr.

The library is created using the ar command from compiled object files.
