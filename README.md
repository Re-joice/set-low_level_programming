# 0x09. Static Libraries

## Description

This project is about creating and using **static libraries in C**.  

We created a static library called `libmy.a` which contains 20 functions, including:

- `_putchar`
- `_islower`
- `_isalpha`
- `_abs`
- `_isupper`
- `_isdigit`
- `_strlen`
- `_puts`
- `_strcpy`
- `_atoi`
- `_strcat`
- `_strncat`
- `_strncpy`
- `_strcmp`
- `_memset`
- `_memcpy`
- `_strchr`
- `_strspn`
- `_strpbrk`
- `_strstr`
